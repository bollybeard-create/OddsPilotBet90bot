# OddsPilotBet90 - Smart Betting Assistant Bot

Your AI-powered betting assistant for odds, predictions, and betting advice.

## Features

- 📊 AI-generated betting odds
- 🎯 Smart betting advice
- 🧠 Game insights and analysis
- 🔮 Match predictions
- 💡 Daily betting tips
- ⚙️ Customizable settings

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/odds` | Get AI-generated odds |
| `/bet` | Get betting advice |
| `/insights` | Game insights |
| `/predict` | Match predictions |
| `/tips` | Daily betting tips |
| `/settings` | Customize settings |

## Deployment on Railway

1. Push this code to GitHub
2. Connect repository to Railway
3. Add environment variables:
   - `TELEGRAM_TOKEN` - Your bot token from @BotFather
   - `OPENAI_API_KEY` - Your OpenAI API key
4. Deploy!

## Environment Variables

| Variable | Description |
|----------|-------------|
| `TELEGRAM_TOKEN` | Telegram Bot Token |
| `OPENAI_API_KEY` | OpenAI API Key |

## Quick Start

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file with your tokens
4. Run: `python bot.py`
